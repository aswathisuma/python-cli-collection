Python CLI Collection
--------------------------------------------------------------------------------------------------------------
A collection of standalone Python command-line projects built for learning, practice, and backend development.

Projects Included
1. Pattern & Math CLI Application
----------------------------------
Menu-driven Python application featuring:
Pyramid star pattern generation
Inverted number pattern
Recursive sum of natural numbers
Lambda-based power calculation

Concepts Used:
Functions
Nested loops
Recursion
Lambda expressions
User input handling

2. Student Record Management System
-------------------------------------
Python program to manage student records and perform data analysis.

Features:
Store student details and marks
Dictionary creation using zip()
String operations on names
List comprehension filtering
Tuple and Set operations
Shallow copy vs Deep copy demonstration
Exception handling for invalid inputs

Concepts Used:
Lists & Nested Lists
Dictionaries
Tuples
Sets
List Comprehensions
Shallow & Deep Copy
Error Handling

3. Student Rank List Management System
----------------------------------------
Object-Oriented Python application for preparing student rank lists.

Features:
Student rank generation
Total and average mark calculation
Grade evaluation
Sports score integration
Sorting students based on rank
Student count tracking

Concepts Used:
OOP Principles
Encapsulation
Inheritance
Multiple Inheritance
Abstraction
Polymorphism
Operator Overloading
Static Methods
Class Methods
